# Contract-Compliance-Monitor
Using AI to get legal guidance from your contract.
Instructions:
  Click the following link:
    https://gistcdn.githack.com/Kron4664/23c7e1057c67ce76a85fb295c0f720ea/raw/afe79baf34d0a481b015053e85618fc971959e1e/contract-compliance-monitor_9.html
  Click the red button saying: "Open the page"
  Go to analyse tab and either upload a contract, past the contract text, or load a sample contract. 
  Click analyze.
  Navigate through sub tabs to see different provisions.
  You can click off anything that has been completed.
  Furthermore, you can use the legal guide for a basic understanding.
